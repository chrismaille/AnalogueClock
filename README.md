# AnalogueClock

A draggable analogue clock addon for World of Warcraft, compatible with ElvUI.

## Features

- **Draggable**: Click and drag the clock to position it anywhere on screen
- **Position persistence**: Your clock position is saved between sessions
- **ElvUI compatible**: Works independently of the minimap, compatible with ElvUI and other UI modifications
- **Latest WoW version**: Updated for The War Within (11.0.5)

## Usage

- **Left-click and drag**: Move the clock to your desired position
- **Left-click**: Open calendar
- **Right-click**: Toggle time manager

## Installation

1. Download the addon
2. Extract to your `World of Warcraft/_retail_/Interface/AddOns/` directory
3. Restart WoW or reload UI with `/reload`

## Compatibility

- World of Warcraft: The War Within (11.0.5)
- Compatible with ElvUI and other UI modifications
- Position is saved in `AnalogueClock_Settings` saved variable
